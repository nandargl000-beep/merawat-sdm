MerawatSDM.id - Paket Website Interaktif (Demo)

Isi paket:
- index.html
- blog.html
- posts/ (artikel1.html, artikel2.html)
- styles.css
- script.js
- images/ (favicon.svg, header.svg, sweet-heat-lightning-instrumental.wav)

Catatan penting:
- Audio file `images/sweet-heat-lightning-instrumental.wav` adalah *placeholder ambient* yang disintesis secara otomatis.
  Jika Anda ingin menggunakan lagu "Sweet Heat Lightning" atau versi instrumental sebenarnya, pastikan Anda memiliki hak atau lisensi.
  Ganti file tersebut dengan file .mp3/.wav yang berlisensi dan pertahankan nama file yang sama (atau ubah src di index.html/blog.html/posts accordingly).
- Untuk meng-host di GitHub Pages:
  1. Buat repo di GitHub dan upload seluruh folder konten.
  2. Di Settings -> Pages, pilih branch main (root) dan simpan.
  3. Akses: https://<username>.github.io/<repo>/

Autoplay notice:
- Beberapa browser memblokir autoplay audio yang tidak dimute. Kode situs mencoba autoplay tapi jika diblokir, pengguna akan melihat prompt kecil untuk memulai audio secara manual.

Hak cipta:
- Semua gambar SVG di folder images adalah milik paket ini (dibuat sederhana untuk tujuan demo).
